# Nesneye-Dayal-Programlama-Odevi-taha-yilmaz
Bu proje, Nesne Yönelimli Programlama (OOP) ve Veritabanı Programlama dersleri kapsamında Java ile geliştirilmiş, konsol tabanlı bir kütüphane yönetim uygulamasıdır.
Temel Özellikler
İşlevsellik: Kitap ve Öğrenci kayıt işlemleri ile Ödünç Verme ve Geri İade (teslim tarihini güncelleme) süreçlerini yönetir.
Veri Kalıcılığı: Veriler, programdan bağımsız olarak SQLite veritabanında kalıcı olarak saklanır.
Mimari: Veri erişimi, temiz ve düzenli bir yapı sağlayan Repository Pattern ile ayrılmıştır.
Teknoloji: Java, SQLite ve JDBC (Java Database Connectivity) kullanılmıştır.
Projeyi çalıştırmak için Java JDK'nın ve SQLite JDBC sürücüsünün projeye dahil edilmiş olması yeterlidir. Uygulama, Main.java dosyası üzerinden başlatıldığında, veritabanı dosyası otomatik olarak oluşturulur.
